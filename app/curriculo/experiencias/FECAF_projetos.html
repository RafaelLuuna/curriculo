1º Semestre

Principal entrega: Planejar e desenvolver uma solução para instituições de ensino.

Para desenvolver esta aplicação, utilizei diversos conceitos da metodologia ágil, tais como:

  • Realizar reuniões constantes com o time para alinhamento
  • Criar wireframes para exemplificar as regras de negócio
  • Controlar demandas através de quadros kanban
  • Utilizar ferramentas como Miro e Trello para auxiliar na gestão.

Também aplicamos conceitos técnicos para a entrega deste projeto, tais como:

  • Criar modelo de banco de dados amplo e escalável, para atender todas as relações necessárias
  • Utilizar um framework full-stack
  • Publicar aplicação na rede

Desenvolver a aplicação não fazia necessáriamente parte do trabalho porém queríamos aproveitar o conhecimento que já tinhamos para rodar a aplicação na prática.

Desenvolvemos a aplicação usando Django com banco de dados SQLite (padrão do Django) e publicamos ela em um sistema Debian utilizando NGINX + Gunicorn.

2º Semestre

Principal entrega: Montar uma máquina completa em um case mode personalizado para rodar um servidor e servir uma aplicação.

Para realizar a entrega deste trabalho, utilizei conceitos de rede e segurança aprendidos durante o semestre como configurações de ip, firewall, conxões remotas e como subir uma aplicação na web.
Aproveitamos a aplicação desenvolvida no semestre anterior para publica-la novamente, aplicando os conceitos aprendidos em aula. Durante a execução do trabalho passamos pelos seguintes processos:

  • Montagem do hardware
  • Instalação do sistema operacional
  • Virtualização da máquina
  • Configuração da rede (para acesso remoto e proteção do servidor)
  • Deploy da aplicação (utilizando nginx + gunicorn no linux, e IIS no Windows Server)
