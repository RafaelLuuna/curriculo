Este projeto trata-se de uma automação para jogar o jogo "double" da blaze.

A idéia do bot é simular rodadas do jogo com uma carteira virtual e utilizar estratégias para maximizar ganhos.

Neste projeto foi utilizado algumas bibliotecas python como Keras para gerar modelos de IA simples e o Numpy para manipulação de arrays. Utillizei também o chromirum para executar comandos pelo navegador.
